# Contributing to DegreeTrackr

## Running tests

```bash
cd frontend && npm test
```

## Type checking

```bash
cd frontend && npm run typecheck
```

## Running Playwright end-to-end tests

```bash
cd frontend && npx playwright test
```

## CI status

GitHub Actions is currently paused due to billing. Use `gh pr merge --admin` to bypass required CI checks when merging pull requests.
